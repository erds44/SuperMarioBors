## Usage

```
run bin\Debug\SuperMarioBros.exe

Keyboard Controller:
  Arrow - left, right, up, down
  x - speed up/fire ball
  q - quit
  r - reset
  p - pause
  O - unpause

GamePad Controller:
  Left Thumb Stick - left, right, up, down
  Right Trigger - speed up/fire ball
  Button A - jump
  Button Y - quit
  Right Shoulder - pause/unpause
```
